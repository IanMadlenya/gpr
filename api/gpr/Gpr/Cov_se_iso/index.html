Module Gpr.Cov_se_iso

Isotropic squared exponential covariance

The covariance is defined as:

k(x, y) = sf^2 * exp(-1/2 * |1/ell*(x-y)|^2)

where sf^2 is the amplitude, and ell is the length scale.
